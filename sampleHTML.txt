<h2>getURLsFromHTML()</h2>
<p><code>getURLsFromHTML(htmlBody, baseURL)</code> takes 2 arguments. The first is an HTML string as we discussed earlier, while the second is the root URL of the website we're crawling. This will allow us to rewrite <a href="https://www.seoclarity.net/resources/knowledgebase/difference-relative-absolute-url-15325/" target="_blank" rel="noopener nofollow">relative URLs into absolute URLs</a>.</p>
<p>It returns an <em>un-normalized</em> array of all the URLs found within the HTML.</p>
<h2>Write some tests before writing your code</h2>
<ul>
<li>Test that relative URLs are converted to absolute URLs.</li>
<li>Test to be sure you find <em>all</em> the <code>&lt;a&gt;</code> tags in a body of HTML</li>
</ul>
<p>Then go ahead and implement the function itself!</p>
